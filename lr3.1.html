Лабораторна робота №3.1

Тема: Функціональне застосування JavaScript у HTML-документі. Використання масивів у JS-сценаріях

Мета роботи: придбати практичні навички роботи з конструкціями мови JavaScript, масивами та функціями у JS-сценаріях. Реалізувати прості програми з використанням базових структур даних.

Завдання №1 — Функції makeTransaction() і getShippingMessage()

Оголошуємо дві функції для розрахунку вартості замовлення та доставки:

function makeTransaction(quantity, pricePerDroid) { const totalPrice = quantity * pricePerDroid; return `You ordered ${quantity} droids worth ${totalPrice} credits!`; } function getShippingMessage(country, price, deliveryFee) { const total = price + deliveryFee; return `Shipping to ${country} will cost ${total} credits`; } console.log(makeTransaction(3, 1500)); console.log(getShippingMessage("Ukraine", 2000, 300));

Результат у консолі:

You ordered 3 droids worth 4500 credits!
Shipping to Ukraine will cost 2300 credits

Завдання №2 — Перевірка повідомлень та транзакцій

Функція для перевірки спам-повідомлень:

function checkForSpam(message) { const lower = message.toLowerCase(); return lower.includes("spam") || lower.includes("sale"); } console.log(checkForSpam("Buy now SALE offer!")); // true console.log(checkForSpam("Your order is confirmed")); // false

Також приклад розрахунку покупки з перевіркою коштів клієнта:

function makeTransaction(quantity, pricePerDroid, customerCredits) { const total = quantity * pricePerDroid; return total > customerCredits ? "Insufficient funds!" : `You ordered ${quantity} droids worth ${total} credits!`; } console.log(makeTransaction(5, 1000, 3000)); // Insufficient funds! console.log(makeTransaction(2, 1000, 3000)); // You ordered 2 droids worth 2000 credits!

Завдання №3 — Масиви та фільтрація

Функція для фільтрації елементів масиву, більших за вказане значення:

function filterArray(numbers, value) { const result = []; for (let number of numbers) { if (number > value) result.push(number); } return result; } console.log(filterArray([1, 5, 10, 15, 20], 8)); // [10, 15, 20]

Функція для злиття двох масивів із обмеженням довжини:

function makeArray(firstArray, secondArray, maxLength) { const newArray = [...firstArray, ...secondArray]; return newArray.length > maxLength ? newArray.slice(0, maxLength) : newArray; } console.log(makeArray([1, 2], [3, 4, 5], 4)); // [1, 2, 3, 4]

Завдання №4 — Обробка масиву

Функція знаходить максимальний серед парних елементів та мінімальний серед парних індексів, потім міняє їх місцями:

function processArray(arr) { let maxEven = -Infinity, minIndex = 0; for (let i = 0; i < arr.length; i++) { if (arr[i] % 2 === 0 && arr[i] > maxEven) maxEven = arr[i]; if (i % 2 === 0 && arr[i] < arr[minIndex]) minIndex = i; } const maxIndex = arr.indexOf(maxEven); [arr[maxIndex], arr[minIndex]] = [arr[minIndex], arr[maxIndex]]; return arr; } console.log(processArray([5, 8, 12, 3, 4, 10]));

Завдання №5 — Калькулятор

Реалізація простого калькулятора на JavaScript:

function calculate(a, b, operator) { switch(operator) { case '+': return a + b; case '-': return a - b; case '*': return a * b; case '/': return b !== 0 ? a / b : "Division by zero!"; case '%': return a % b; case '^': return Math.pow(a, b); default: return "Unknown operator"; } } console.log(calculate(5, 3, '+')); // 8 console.log(calculate(10, 2, '^')); // 100
