Завдання №1 — Розробка макету сайту

Було створено макет сайту магазину комп’ютерної техніки. Використано блокову верстку на основі Flexbox. Основна концепція — розподіл елементів у контейнерах для адаптивного відображення.

HTML-код макету:

<header> <h1>Магазин комп’ютерної техніки</h1> </header> <nav> <a href="#">Головна</a> <a href="#">Каталог</a> <a href="#">Контакти</a> </nav> <main> <section>Товари</section> <aside>Фільтри</aside> </main> <footer>© 2025 Комп'ютерний магазин</footer>

CSS-код:

body { display: flex; flex-direction: column; min-height: 100vh; } main { display: flex; flex: 1; } section { flex: 3; background: #11131b; padding: 10px; } aside { flex: 1; background: #1a1c24; padding: 10px; }

Завдання №2 — Таблична та блокова верстка

Порівняння типів макетів:

  • Таблична верстка: проста у реалізації, але незручна для адаптивного дизайну.
  • Блокова верстка: гнучка, легше адаптується під різні екрани.
  • Flexbox: сучасний підхід, зручне вирівнювання елементів у ряд і колонку.

Завдання №3 — Верстка за допомогою Flexbox

У цьому завданні створено макет сторінки із секцією товарів, розміщених за допомогою Flexbox. Кожен блок автоматично підлаштовується під розмір екрана.

.container { display: flex; flex-wrap: wrap; justify-content: space-between; } .product { flex: 0 1 30%; background: #161821; border: 1px solid #00fff5; margin: 10px; border-radius: 6px; padding: 10px; color: #fff; }

Висновки

У результаті виконання роботи я навчився створювати макети веб-сторінок різних типів: табличні, блокові та на основі Flexbox. Flexbox забезпечує просту адаптивність і контроль над вирівнюванням елементів, що робить його ефективним інструментом сучасної верстки.
