Лабораторна робота №1.2

Тема: Каскадні таблиці стилів. Селектори. Ідентифікатори. Стильове оформлення текстових елементів

Мета роботи: придбати практичні навички роботи з селекторами тегів, класів, ідентифікаторів, списками, властивостями кольору, фону, оформленням текстових елементів у HTML-документах.

1. Селектори тегу

Селектори тегів застосовуються безпосередньо до всіх елементів одного типу.

p {
color: #00fff5;
font-size: 18px;
}

Це приклад тексту, оформленого через селектор тегу <p>.

2. Селектори класу

Клас дозволяє задавати унікальне оформлення для групи елементів.

.highlight {
color: #ffeb3b;
background-color: #333;
padding: 5px;
border-radius: 5px;
}

Цей текст оформлено за допомогою класу .highlight

3. Селектори ідентифікатора

Ідентифікатор використовується для одного унікального елемента сторінки. Позначається знаком решітки #.

#special-text {
color: #00ff9d;
font-weight: bold;
text-decoration: underline;
}

Цей абзац має унікальний стиль через ID-селектор.

4. Інші селектори

Можна використовувати дочірні, сусідні, атрибутні та універсальні селектори.

div > p { color: #ff9800; }
input[type="text"] { background: #1e1e1e; color: white; }
* { font-family: 'Poppins', sans-serif; }
